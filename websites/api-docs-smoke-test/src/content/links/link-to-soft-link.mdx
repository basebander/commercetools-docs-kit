# Linking from Markdown

Example:

```md
[ArrayTestType](/types/general#ctp:test:type:ArrayTestType)
```

Rendered link: [ArrayTestType](/types/general#ctp:test:type:ArrayTestType)

# Linking from Description of a Type

Example:

```yaml
LinkFromDescription:
    description: |-
      Link [ArrayTestType](/types/general#ctp:test:type:ArrayTestType) with `[ArrayTestType](/types/general#ctp:test:type:ArrayTestType)`
    properties:
      linkFromPropertyDescription:
        description: |-
          From property description, link [ArrayTestType](/types/general#ctp:test:type:ArrayTestType) with `[ArrayTestType](/types/general#ctp:test:type:ArrayTestType)`
        type: string
      linkWithMicrositeNotation:
        description: |-
          Link [NonExistentType](/../non-existent-microsite/types/non-existent-page#ctp:non-existent-api:type:NonExistentType) with `[NonExistentType](/../non-existent-microsite/types/non-existent-page#ctp:non-existent-api:type:NonExistentType)`
        type: string
      linkWithAbsoluteURL:
        description: |-
          Link [NonExistentType](https://docs.commercetools.com/non-existent-microsite/types/non-existent-page#ctp:non-existent-api:type:NonExistentType) with `[NonExistentType](https://docs.commercetools.com/non-existent-microsite/types/non-existent-page#ctp:non-existent-api:type:NonExistentType)`
        type: string
```

Rendered type:

<ApiType apiKey="test" type="LinkFromDescription" />
