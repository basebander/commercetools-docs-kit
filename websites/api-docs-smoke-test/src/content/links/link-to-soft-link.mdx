URN patterns for soft links:

* Type: ctp:{api-key}:type:{type-name}
* Resource:

# Linking from Description of Type or Resource

To link properly from the a type or a resource, always use full path or with an absolute URL

Link with
* full path
* absolute url
* microsite notation
